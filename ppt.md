## Slide 10: Security & Architecture
**Heading:** Secure by Design
**Points:**
- **PostgreSQL RLS:** Row Level Security ensures users only access their own data.
- **Supabase Auth:** Enterprise-grade security (Email, Magic Links, OAuth).
- **Encrypted Storage:** Resumes and sensitive media are stored in secure buckets.
- **Middleware Protection:** Every private route is guarded by auth middleware.
